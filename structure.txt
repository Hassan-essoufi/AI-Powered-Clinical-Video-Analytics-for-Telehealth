│   ├── ai_modules/
│   │   ├── rppg/

│   │   ├── evm/

│   ├── services/

the  architecture : 

client Camera (Browser)
       │
       ▼
   WebRTC Stream
       │
       ▼
Backend Server (Python + FastAPI + aiortc)
       │
       ├── Frame Receiver (VideoTrack)
       │
       ├── Frame Queue (asyncio / multiprocessing)
       │
       ├── AI Modules (run in parallel):
       │      ├─ rPPG Module (Heart Rate, Respiration)
       │      ├─ EVM Module (Pulse / Micro-movement visualization)
       │      └─ Wound Measurement Module (Click-to-measure)
       │
       └── API & WebSocket Server
                │
                ▼
        Doctor Dashboard (Browser)
        ├─ Video display
        ├─ Toggle “Magnify” (EVM)
        ├─ Display Heart Rate / Respiration
        └─ Click-to-measure wounds
